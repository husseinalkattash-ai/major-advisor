# Majors Knowledge Base

For each major: its ideal profile (the `requirement` facts in
`src/requirements.clp`) and the rationale for every requirement and weight.
Levels are on the shared ordinal scale (academic: weak < average < good <
excellent; interest/skill: low < medium < high). Weight is 1-3, expressing
how decisive the attribute is for the major. `max weight` is the sum of a
major's weights -- the ceiling a perfectly-matched student would score.

RIASEC code legend: R realistic, I investigative, A artistic, S social,
E enterprising, C conventional.
